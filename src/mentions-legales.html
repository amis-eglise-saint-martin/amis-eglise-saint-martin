Protection des données personnelles

Responsable du traitement

L'association des Amis de l'Église Saint-Martin est responsable du traitement des données personnelles collectées sur ce site.

Données collectées

Les seules données personnelles collectées sont celles que vous nous transmettez volontairement via le formulaire de contact :

  • Nom et prénom
  • Adresse email
  • Numéro de téléphone (optionnel)
  • Contenu de votre message

Finalité et base légale

Ces données sont utilisées uniquement pour répondre à vos demandes. La base légale est votre consentement, exprimé lors de l'envoi du formulaire.

Destinataires

Vos données sont transmises à :

  • Formspree (traitement du formulaire) - Politique de confidentialité
  • Les membres du bureau de l'association, pour répondre à votre demande

Conservation

Les données du formulaire sont conservées le temps nécessaire au traitement de votre demande, puis supprimées dans un délai maximum de 3 ans.

Vos droits

Conformément au RGPD, vous disposez des droits suivants :

  • Accès : obtenir une copie de vos données
  • Rectification : corriger des données inexactes
  • Effacement : demander la suppression de vos données
  • Opposition : vous opposer au traitement

Pour exercer ces droits, contactez-nous ou écrivez à notre adresse postale.

En cas de litige, vous pouvez saisir la CNIL.
